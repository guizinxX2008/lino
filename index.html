Jornada ao Tesouro Perdido no Egito

Passo 1: Início da Jornada

Você está em Cairo, no Egito. Você encontrou um antigo pergaminho com uma pista sobre o tesouro perdido.

Passo 2: Escolha da Rota

O pergaminho indica dois caminhos: um através do deserto e outro pelas catacumbas.

Passo 3: Desafios do Deserto

Você entrou no deserto e encontrou um antigo acampamento. Os habitantes oferecem duas opções: uma rota segura ou um atalho arriscado.

Passo 3: Catacumbas

Você optou pelas catacumbas e encontrou uma entrada para uma tumba. Dentro, você se depara com duas opções: um corredor estreito ou uma passagem secreta.

Passo 4: Rota Segura

A rota segura leva você a uma antiga pirâmide. Dentro da pirâmide, você encontra dois caminhos: um para o salão principal e outro para uma câmara oculta.

Passo 4: Atalho Arriscado

O atalho arriscado leva você a uma oásis escondido. Você encontra um antigo mapa com uma pista para o tesouro.

Passo 4: Corredor Estreito

O corredor estreito leva a uma sala com um enigma. Resolver o enigma revela a localização do tesouro.

Passo 4: Passagem Secreta

A passagem secreta leva você a uma sala cheia de hieróglifos antigos. Você deve decifrar os hieróglifos para avançar.

Passo 5: Salão Principal

No salão principal, você encontra três sarcófagos: um dourado, um prateado e um de bronze.

Passo 5: Câmara Oculta

Na câmara oculta, você encontra uma caixa antiga com um enigma que revela uma pista final.

Passo 5: Mapa do Oásis

Seguindo o mapa, você encontra uma entrada secreta para uma antiga tumba.

Passo 5: Enigma do Corredor

Resolver o enigma revela a localização de uma entrada secreta para o tesouro.

Passo 5: Hieróglifos

Decifrar os hieróglifos revela um mapa para uma tumba escondida.

Passo 6: Sarcófago Dourado

O sarcófago dourado leva a uma sala cheia de ouro, mas o verdadeiro tesouro está escondido atrás de uma parede secreta.

Passo 6: Sarcófago Prateado

O sarcófago prateado contém um enigma que, se resolvido, revela o caminho para o tesouro.

Passo 6: Sarcófago de Bronze

O sarcófago de bronze contém um mapa que leva ao verdadeiro tesouro.
